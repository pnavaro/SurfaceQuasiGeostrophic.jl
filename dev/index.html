SurfaceQuasiGeostrophic

Documentation for SurfaceQuasiGeostrophic.

The unknown of the model is the buoyancy $b(t,x,y)$ and the stream function $\psi(t,x,y)$ which are related through a Poisson like equation $| −i \nabla | \psi = b$, which becomes in Fourier $|k| \hat{\psi} = \hat{b}$

The dynamical model is the following

\[\frac{\partial b}{\partial t} + (\nabla \times \psi) \cdot \nabla b = 0. \]

The solver uses Fourier techniques so that

\[\nabla \times \psi \cdot \nabla b = ifft[(ik) \times \hat{\psi}] \cdot ifft[(ik)\hat{b}],\]

Derived type to solve the Poisson equation on 2d regular cartesian mesh with periodic boundary conditions on both sides

  • kx : wave number in x
  • ky : wave number in y
  • k2 : $k_x^2 + k_y^2$
  • nc_x : cells number in x
  • nc_y : cells number in y
  • dx : x step size
  • dy : y step size
